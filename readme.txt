== Description ==
This plugin adds a an icon for Opera Speed Dial

To make this plugin work, upload an icon, saved as PNG as: wp-content/osd-image.png
Else this plugin won`t work!

== Installation ==
1. Upload `opera-speed-dial/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
